1. 둘러보기
    1.1 eidit 모드와 command 모드
    - command: 키보드 단축기를 통해 전반적인 셀을 관리(파랑)
    - eidit: 하나의 셀에 입력하거나 관리할 수 있음(초록)
    
    1.2 단축키
    - 나머지 단축키는 키 옆에 쓰여있으니 참고
    - dd: 셀 삭제
    - ctrl + enter: 셀 실행
    - b: 선택된 셀 밑에 새로운 셀 생성
    - a: 선택된 셀 위에 새로운 셀 생성

    1.3 작동 순서
    - ln[숫자]가 셀 옆에 써있는데 이 숫자가 실행된 순서
    - 즉, 위에서 아래로 차례대로 진행되는 것이 아니다
    - run을 하면 숫자는 초기화되지 않고 계속해서 이어서 진행됨
    - 처음부터 다시 하고 싶으면 'restart kernel and re-run'을 실행

    1.4 Markdown
    - '4. Markdown 연습.ipynb' 참고

    1.5 셀 출력
    - print를 쓰면 모두 출력
    - 그러나 셀의 마지막 줄은 print가 없어도 출력
    - print가 없으면 데이터에 맞춰서 출력해주기도 함
