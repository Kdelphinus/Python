1. HTML 태그의 구성
    1.1 태그 이름
    - p, li, img 등
    - <>안에 들어가있다
    - <p>, <li>, <img> 등

    1.2 속성
    - 모든 HTML태그는 속성이라는 추가 정보를 가질 수 있다
    - 태그 이름이 아닌 것은 모두 속성이라고 볼 수 있다
    - 속성은 일반적으로 속성이름과 값을 하나의 쌍으로 가진다
    - <li id="favorite">우유</li>
        - 위 <li> 태그는 id라는 속성이 있고 그 값은 "favorite"이다
    - <img alt="logo" class="logo-img" src="/images/music/logo.png"/>
        - 한 태그가 여러 속성을 가질 수도 있다

2. HTML 태그의 구조
<ul>
    <li>커피</li>
    <li>녹차</li>
    <li>우유</li>
</ul>
- <ul> 태그가 부모, <li> 태그가 자식 태그
- <ul> 태그 아래에 세 개의 <li> 태그가 있고, 그 안에 데이터가 있다

3. HTML을 얼마나 알아야 할까?
- 웹사이트에서 데이터를 모을 수 있는 정도가 목표라면 태그의 구조를 이해하는 수준이면 됨
- 기본적 데이터 수집을 넘어 자동화 봇, 복잡한 사이트의 크롤링 등을 하려면 HTML과 CSS를 능숙하게 다뤄야 함

4. 기본 HTML 태그 정리
- 2. index.html 참고
    4.1 DOCTYPE 선언
    - <!DOCTYPE html>
    - 가장 먼저 선언해야 하는 것

    4.2 title 태그
    - <title>Sample Website</title>
    - 브라우저의 탭이나 방문 기록에 나와 있는 제목이 여기 들어감

    4.3 h1~h6 태그
    - <h1>머리말 1</h1>
    - <h2>머리말 2</h2>
    - <h3>머리말 3</h3>
    - <h4>머리말 4</h4>
    - <h5>머리말 5</h5>
    - <h6>머리말 6</h6>
    - 한 페이지에 여러 개의 머리말이 있을 수 있다
    - 가장 중요한 머리말은 1번, 그 다음으로 중요한 머리말은 2번 ... 6번까지 가능
    - 크기는 따로 설정이 가능하지만 따로 설정하지 않으면 1번부터 순서대로 작아짐

    4.4 p태그
    - <p> ... </p>
    - 보통 문단이 이 안에 들어감
    - p태그 위, 아래에는 기본적으로 여백이 조금씩 있다
