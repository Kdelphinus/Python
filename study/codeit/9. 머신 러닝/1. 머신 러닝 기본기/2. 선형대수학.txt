1. 개요
- 일차식이나 일차 함수를 공부하는 학문

2. 행렬과 벡터
- 벡터: 열이 하나만 있거나 행이 하나만 있는 행렬
- 일반적으로 열벡터를 많이 쓰기에 벡터라고 부르고 행 벡터를 쓸 땐, 행 벡터라고 이야기한다
- 행렬은 대문자, 벡터는 소문자를 사용하기로 한다

3. 행렬 연산
    3.1 행렬 덧셈
    - 같은 위치의 값들을 더하면 된다
    - 더하는 행렬은 같은 크기를 가져야 한다

    3.2 행렬의 곱
        3.2.1 스칼라곱
        - 행렬과 스칼라가 곱할 땐, 각 항목에 스칼라를 곱하면 된다

        3.2.2 두 행렬의 곱(여기선 내접곱만)
        - A x B = AB일 때
        - A의 1행과 B의 1열을 각각 곱한 값들의 합이 AB의 1행 1열에 들어간다
        - 이렇게 진행하면  A의 행과 B의 열을 가진 AB가 나온다
        - 그렇기에 A의 열과 B의 행의 수가 같아야 곱할 수 있다
        - A(m x n) x B(n x p) = AB(m x p)
        - 곱하는 순서에 따라서도 결과가 바뀜

        3.2.3 요소별 곱하기
        - 같은 크기를 가지는 행렬 두 개가 같은 위치의 요소끼리 곱하여 새로운 행렬을 만드는 것
        - 표기는 동그라미(o)를 써서 표현: A o B
        - numpy에서 A * B하면 요소별 곱하기가 됨

4. 특수 행렬
    4.1 전치 행렬
    - 특정 행렬의 열과 행을 바꾸는 것
    - 행렬 위의 T를 붙임

    4.2 단위 행렬
    - 대각선으로는 1, 나머지는 0
    - 크기는 항상 정사각형으로 주어짐
    - 단위 행렬과 곱하면 어떤 행렬이든 같은 값이 나옴

    4.3 역행렬
    - 특정 행렬에 역행렬을 곱하면 단위 행렬이 나와야 한다
    - 모든 행렬에 역행렬이 있지는 않다
    - 정사각형으로 주어진다

5. 선형대수학이 머신러닝에 필요한 이유
- 아파트 가격 예측의 경우 크기, 지하철 역 거리, 층수에 영향을 받는다면
- 관계의 식들을 행렬로 이용하여 계산하거나 정리하기가 쉽다
- [아파트의 종류 x 아파트의 특성]의 행렬과 [특징 영향력]의 벡터로 쓸 수 있다

- 머신 러닝에선 데이터를 일차식에 이용하는 경우가 많고 행렬을 이용하면 그것을 정돈된 형태로 효율적 계산이 가능하기에 선형대수학을 사용
