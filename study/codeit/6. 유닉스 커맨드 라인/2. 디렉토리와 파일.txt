1.경로
- 현재 사용자: 지금 컴퓨터를 사용하고 있는 사용자
- 홈 디렉토리: 자신이 필요한 것을 저장하는 자신의 디렉토리
- ~: 사용자의 홈 디렉토리를 의미, 현재 위치가 바뀔 때마다 이 부분이 변함
- pwd: 홈 디렉토리의 위치를 출력
- cd 가고싶은경로: 디렉토리를 변경
- ls: 자식 디렉토리들을 출력
- cd /: 루트 디렉토리로 이동
- cd ~: 홈 디렉토리로 이동
- cd -: 이전 디렉토리로 이동

2. 절대 경로와 상대 경로
- 절대 경로: 루트 디렉토리를 기준으로 어떤 파일이나 디렉토리가 가지고 있는 고유한 경로
    예시) /Home/delphinus
- 상대 경로: 나의 현재 위치를 기준으로 나타낸 경로
    .: 현재 위치한 디렉토리(pwd를 입력할 때, 나오는 위치와 동일)
    ..: 현재 디렉토리를 포함한 부모 디렉토리
    현재 경로가 루트 디렉토리에서 멀리 있는 상황에서 주변 경로를 나타낼 때
    혹은 프로그램의 호환성을 좋게 할 때(설치한 파일보다 상위 폴더는 각 환경마다 다를 수 있기에 절대경로로 지정하면 에러가 뜰 확률이 높음)
    상대 경로를 사용
    
3. 디렉토리 내부 살펴보기
- ls: 하위 폴더를 보여줌
- ls -l: 결과를 자세한 정보를 보여줌
- ls -a: 숨겨져 있는 자식 디렉토리나 파일을 볼 수 있음(폴더이름이 .으로 시작한 것은 ls로 나오지 않음, 숨겨진 파일)

    3.1 인자
    - ls -l 파일이름: 파일이름을 인자로 주면 그것에 대한 설명이 나옴, 폴더를 넣으면 그 폴더에 하위 폴더가 나옴
    - ls -l -d 폴더이름: 폴더 자체의 정보를 보여줌
    
    3.2 결과
    - ls로 부른 파일들 이름의 앞은 파일 종류를 알려줌
    - '-': 일반파일, d: 디렉토리, c: 캐릭터 장치 파일, b: 블록 장치 파일, s: 소켓, p: 파이프, l: 심볼릭 링크

4. 디렉토리와 파일 만들기
- mkdir '만들고 싶은 디렉토리 이름': 현재 위치에 디렉토리가 생성
- mkdir '만들고 싶은 디렉토리 경로': 경로가 지정된 곳에 디렉토리 생성
- touch 파일이름: 파일 생성
- 여러 개의 파일이나 폴더를 한 번에 생성할 수 있음

5. 디렉토리와 파일 옮기기
- mv '파일이름' '옮길디렉토리이름'
- mv '폴더이름' '경로'
- mv '파일이름' '새로운이름'
- mv '폴더이름' '새로운이름'
- 같은 이름의 파일이 새로운 경로에 있다면 그것을 덮어 씌어버림
- 그것을 방지하기 위해 mv -i '파일이름' '경로'를 써서 동일 이름이 있을 때, 덮어 쓸 것인지 확인하게 함
- y는 덮어씀, n은 옮기지 않음

6. 디렉토리와 파일 복사 붙여녛기
- cp '원본이름' '복사본이름': 파일만 가능
- -i는 cp에서도 사용가능, 같이 사용하기를 추천
- cp -r '원본폴더이름' '복사본폴더이름': -r은 재귀를 뜻하고 폴더 안에 있는 자료들도 같이 복사하도록 명령

7. 디렉토리와 파일 삭제
- rm '삭제할 파일의 이름'
- rm -r '삭제할 폴더의 이름'
- rm -i '삭제할 폴더의 이름': 디렉토리 안으로 들어갈 것인지 먼저 물어봄, 그리고 파일 하나씩 삭제 여부를 물어봄
                             마지막으로 폴더의 삭제 여부를 물어보지만 파일 중 지우지 않는 것이 있으면 에러가 뜨며 폴더 자체가 삭제되지 않음

8. 파일 내용 출력
- cat '파일이름'
- less '파일이름': 화살표에 따라 한줄씩 이동하고 g는 처음으로 G는 마지막으로 이동, q는 나가는 것, b는 전 페이지, 스페이스바는 다음 페이지
- cat에 파일을 두 개 주면 쭉 붙어서 나옴
- less에 파일 두개 이상을 주면 파일 하나만 우선 나옴
  :n을 누르면 다음 파일로, :p를 누르면 전 파일로 이동
- head '파일이름': 파일의 처음 10줄을 출력
- head -n 20 '파일이름': 처음 20줄을 출력
- tail '파일이름': 뒷 부분 10줄
- tail -n 30 '파일이름': 뒷 부분 30줄

9. 커맨드 사용에 유용한 것들
- history: 지금까지 실행한 커맨드들이 모두 보임, 같은 것을 다시 실행하고 싶으면 '!"커맨더 옆 번호"'를 실행하면 됨
- tab키: 자동완성, 현재 디렉토리 안에 입력한 값으로 시작하는 파일이나 폴더의 이름을 자동완성
         중복값이 있으면 값이 안나옴, 이럴 땐 탭을 두번 누르면 중복되는 값들의 이름을 보여줌
         이름을 더 적고 탭을 누르면 자동완성
- ctrl + a: 커서가 맨 앞으로 이동
- ctrl + e: 맨 뒤로 이동
- clear: 화면을 깨끗하게 함
- 공백이 있는 파일이나 폴더를 만들거나 사용할 때 ''로 감싸주면 됨
